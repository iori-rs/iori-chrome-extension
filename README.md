# Iori Chrome Extension

A Chrome Extension designed to detect and capture media streams (HLS/m3u8 and DASH/mpd) from web pages. It is specifically built to generate download commands for the **Shiori** CLI tool.

## Tech Stack

- **Framework**: [Plasmo](https://docs.plasmo.com/) (Manifest V3)
- **UI**: React 18 + TypeScript
- **Icons**: Lucide React
